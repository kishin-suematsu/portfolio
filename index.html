Projects

I developed an English reading application designed to help learners read from start to finish. In some countries, people are taught to read from end to start, which is not efficient for learning English. This application presents students with an English sentence and multiple related questions. The sentence gradually disappears from the beginning, prompting users to read and understand it before it vanishes completely. Additionally, users can create their own accounts, and the software provides them with recommended questions based on their English proficiency level. As they improve, they can progress to more advanced questions.

I've developed a game called Simon, which challenges memory and concentration. I utilized jQuery along with standard HTML, CSS, and JS to build this game. Once the user starts by typing a key, the game begins. In each round, one of four colored boxes flashes. The player must remember the sequence and click the exact sequence correctly each round. Making a mistake results in a game over message being displayed.
